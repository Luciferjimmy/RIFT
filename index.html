Frequently Asked Questions

Do I need to own the game already?

Yes. RIFT runs games you already own on Steam or Epic. We don't sell game licenses.

Will online multiplayer work?

Singleplayer games work great. Online multiplayer is blocked by anti-cheat on most competitive games — we're transparent about this on every game page.

Is this legal?

Yes. RIFT uses Wine, an open source compatibility layer. Same technology CrossOver has sold commercially for years.

Which Macs are supported?

Apple Silicon only — M1, M2, M3, M4 and all Pro/Max/Ultra variants.

How is this different from CrossOver?

CrossOver is a general solution. RIFT optimizes every game individually — pre-compiled shaders, custom settings, tested fixes. Less work for you, better performance per game.
